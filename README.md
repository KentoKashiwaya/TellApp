# TellApp
電話営業のリストと荷電結果をダッシュボードで管理できるサービスです。<br >
業種ごとのアポ取得効率を確認でき、生産性の向上につながります。<br >
ログイン不要ですぐにお使いいただけます。<br >

- 入力済みデータをチャートへ即時反映
